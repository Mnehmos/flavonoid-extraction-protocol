4. Spectroscopic Elucidation Stack

NMR Sequence

  • 1D: ¹H → ¹³C
  • 2D: COSY → HSQC → HMBC → NOESY
  • J-coupling targets: ortho/meta for flavonoid core

HRMS Interpretation

  • Adduct checklist: [M+H]+, [M+Na]+, [M-H]-
  • RDBE logic: double bond equivalents for scaffold class

UV/CD Diagnostic Rules

  • Flavone: λmax 270, 340 nm; strong CD at 320 nm
  • Flavonol: λmax 260, 370 nm; CD sign inversion at 350 nm
